# SHOP.CO

Магазин одежды с корзиной

# Технологии

- Next.js
- Redux Toolkit
- RTK Query
- TypeScript 
- Node.js (express)

# Локальный запуск 

- ```git clone https://github.com/IlyaVafin/shop.co.git```
- Запуска front части: ```npm run dev```
## Запуск backend
- ```cd backend``` 
- ```node main.js```
